# iOS Shortcut Setup — Inkcast

Three configurations, from simplest to most powerful.

---

## Shortcut 1 — Quick Launch (Recommended)

Opens Inkcast in Safari with one tap. Add to Home Screen for an app-like experience.

**Setup:**
1. Open **Shortcuts** app → tap **+**
2. Search for **Open URLs** → add it
3. Paste: `https://chizkidd.github.io/inkcast/`
4. Tap the shortcut name → rename to **Inkcast**
5. Tap the icon → choose 📖 or 🎙️
6. Tap **Done**
7. Long-press the shortcut → **Add to Home Screen**

---

## Shortcut 2 — Siri Launch

Say **"Hey Siri, Inkcast"** to open hands-free.

Same setup as above. Make sure the shortcut name matches exactly what you want to say.
Good names: `Inkcast`, `Open my books`, `Read to me`

---

## Shortcut 3 — Lock Screen / Control Center

1. Go to **Settings → Control Center** → add a Shortcuts widget
2. Or **Settings → Lock Screen** → add a Shortcuts widget
3. Select your Inkcast shortcut

---

## Getting books into Inkcast

1. Save epub/pdf files to the **Files app** (iCloud Drive recommended)
2. Create a folder called `Inkcast` in iCloud Drive to keep them organised
3. Open Inkcast → tap the upload zone → Files picker opens → navigate to your folder → tap a book

The Files picker remembers your last location, so after the first time it opens straight to your books.

## Tips

**Best voices for listening:**
Go to **Settings → Accessibility → Spoken Content → Voices → English** and download:
- **Siri Voice 1 or 2** (most natural)
- **Daniel** (British, excellent for fiction)
- **Karen** (Australian, clear and warm)

Downloaded voices appear in Inkcast's voice picker automatically under the correct region group.

**Speed preference** is saved automatically and persists across sessions.

## Troubleshooting

| Problem | Fix |
|---|---|
| "Speech not supported" | Use Safari, not an in-app browser (Instagram, Gmail etc.) |
| Book won't parse | Confirm the file opens in another reader. Some DRM-protected files can't be read. |
| Blank page on open | Wait 2 minutes after first GitHub Pages setup, then refresh. |
| Voice says "loading..." | Tap "change voice" — voices load after first user interaction on iOS. |
| Audio stops after a minute | Enable Wake Lock: tap play, then keep screen on. Or download a Siri voice which is more reliable. |
